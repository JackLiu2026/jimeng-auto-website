## 功能模块

### 1. 产品搜索功能
- **实时搜索**: 输入时实时显示匹配结果
- **多条件筛选**: 品牌、分类、价格范围、库存状态
- **模糊匹配**: 支持部分型号匹配
- **排序功能**: 价格、库存量、更新时间
- **搜索历史**: 记录最近搜索

### 2. 库存管理系统
- **自动更新**: 每天4次定时更新
- **Excel导入**: 从AI-STORE文件夹读取Excel
- **数据验证**: 检查数据格式和完整性
- **历史记录**: 保留库存变更记录

### 3. 多语言支持
- **语言切换**: 右上角中/EN切换按钮
- **独立内容**: 中英文内容分别管理
- **自动检测**: 根据浏览器语言自动跳转
- **SEO优化**: 中英文关键词分别设置

### 4. 响应式设计
- **移动优先**: 优先适配手机端
- **断点设计**: 手机、平板、桌面三种布局
- **触摸优化**: 大按钮、易点击
- **性能优化**: 图片懒加载、代码分割

## 技术实现

### 前端技术
- **HTML5**: 语义化标签
- **CSS3**: Flexbox + Grid布局
- **JavaScript**: 原生JS + 少量jQuery
- **搜索算法**: 客户端搜索 + 服务端索引

### 后端服务
- **托管**: GitHub Pages (免费)
- **构建**: Jekyll 静态网站生成器
- **自动化**: GitHub Actions
- **CDN**: Cloudflare (免费加速)

### 第三方服务
- **表单处理**: Formspree (免费版)
- **统计分析**: Google Analytics (免费)
- **搜索增强**: Algolia (可选，有免费额度)

- **主色**: 工业蓝 `#0056b3` (专业可靠)

- **阴影**: 轻微阴影提升层次感
