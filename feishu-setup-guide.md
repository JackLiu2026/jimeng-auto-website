# 飞书应用配置步骤指南

## 第一步：访问飞书开放平台
1. 打开浏览器，访问：https://open.feishu.cn/
2. 使用你的飞书账号登录

## 第二步：创建企业自建应用
1. 点击页面右上角的"创建企业自建应用"
2. 填写应用信息：
   - **应用名称**：OpenClaw文件助手
   - **应用描述**：通过飞书发送文件到电脑
   - **应用图标**：可选，可以上传一个图标
3. 点击"确定创建"

## 第三步：获取应用凭证
创建成功后，在应用详情页面找到：
1. **App ID** - 应用唯一标识（一串数字）
2. **App Secret** - 应用密钥（点击"显示"查看）

## 第四步：配置事件订阅
1. 在左侧菜单点击"事件订阅"
2. 点击"添加事件"
3. 设置请求地址（先留空，稍后填写）
4. 添加以下权限：
   - `im:message` - 接收消息
   - `im:message.p2p_msg` - 接收单聊消息
   - `file:file_uploaded` - 文件上传事件
5. 点击"保存"
6. 获取 **Verification Token**（验证令牌）

## 第五步：保存凭证到桌面
将以下信息保存到桌面文件 `feishu-credentials.txt`：
```
App ID: [你的App ID]
App Secret: [你的App Secret]
Verification Token: [你的Verification Token]
创建时间: [当前日期时间]
```

## 第六步：发布应用
1. 在左侧菜单点击"版本管理与发布"
2. 点击"创建版本"
3. 填写版本信息
4. 申请发布
5. 等待审核通过（企业自建应用通常立即生效）

## 注意事项
1. 保存好所有凭证，不要泄露给他人
2. App Secret 只显示一次，请立即保存
3. 验证令牌用于验证请求来源
4. 应用发布后才能正常使用
