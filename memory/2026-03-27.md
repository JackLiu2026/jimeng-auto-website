## 常州吉盟机电控制系统有限公司外贸网站项目

### 项目概述
为常州吉盟机电控制系统有限公司建设了一个针对东南亚市场的外贸网站，展示施耐德、西门子、SICK等品牌自动化产品的现货库存。

### 已完成工作
1. **库存数据处理**
   - 从桌面AI-STORE文件夹读取Excel文件（SICK.xls, 施耐德.xls, 西门子.xls）
   - 使用Python pandas处理数据，转换为JSON格式
   - 创建了库存数据文件：`_data/inventory.json`

2. **网站架构**
   - 使用Jekyll静态网站生成器
   - 创建了响应式布局设计
   - 实现了中英文双语支持

3. **页面建设**
   - 首页：公司介绍、核心产品展示
   - 库存页面：按品牌分类展示产品，支持搜索
   - 联系页面：多种联系方式（WhatsApp, Zalo, Messenger, 邮箱, 电话）
   - 关于页面：公司详细介绍、服务内容

4. **自动化功能**
   - GitHub Actions工作流自动更新库存数据
   - 每天4次自动同步（8:00, 12:00, 15:00, 17:00 北京时间）
   - 从本地Excel文件自动读取最新库存

5. **技术特点**
   - 完全免费托管（GitHub Pages）
   - 全球CDN加速（Cloudflare）
   - 响应式设计，适配手机、平板、电脑
   - SEO优化，针对外贸关键词
